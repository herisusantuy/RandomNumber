# guess_the_flag
Guess The Flag is simply app built using Swift. Following tutorial from Hacking with Swift by Paul Hudson.
In this app, i learn about : 
 - Interface Builder
 - Auto Layout
 - Outlets
 - @2x and @3x
 - Assets catalogs
 - UIButton
 - CALayer
 - UIColor
 - Random numbers
 - Actions
 - UIAlertController
